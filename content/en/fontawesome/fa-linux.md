## `Linux` font awesome icon CSS class name

To display `Linux` font awesome icon, add predefined class name i.e.,`fa-linux` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Linux` icon.

And use class `fa-inverse` for an alternative icon color.
