## `Otter` font awesome icon CSS class name

To display `Otter` font awesome icon, add predefined class name i.e.,`fa-otter` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Otter` icon.


`Otter` icon has 1 icon style i.e.,solid. 
 We need to append icon style class `fas`.
```
<i class='fas fa-otter'></i>

```

## `Otter` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Otter` icon and `home` icon may not have same height and width.

So to display two icons with fixed width and height we can use `fa-fw` class.
```
<i style='border:1px solid;' class='fas fa-otter fa-fw fa-3x'></i>Fixed Width<i style='border:1px solid;' class='fas fa-otter fa-3x'></i>Normal<br/>
```

And use class `fa-inverse` for an alternative icon color.
