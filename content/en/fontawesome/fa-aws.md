## `Aws` font awesome icon usage
We can display `Aws` font awesome icon using 3 different ways depending upon our requirement.

1. Using CSS Class Name 

2. Using CSS Content Code 

3. Using SVG 



## `Aws` font awesome icon CSS class name

To display `Aws` font awesome icon, add predefined class name i.e.,`fa-aws` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Aws` icon.


`Aws` icon has 1 icon style i.e.,brands. 
 We need to append icon style class `fab`.
```
<i class='fab fa-aws'></i>

```

## Combine `Aws` font awesome icon with other font awesome icons
