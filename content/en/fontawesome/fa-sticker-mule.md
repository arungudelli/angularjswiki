## `Sticker-Mule` font awesome icon CSS class name

To display `Sticker-Mule` font awesome icon, add predefined class name i.e.,`fa-sticker-mule` (with prefix `fa-`) to the `i` tag.

`Sticker-Mule` icon has 1 icon style i.e.,brands. 
 We need to append icon style class `fab`.

And use class `fa-inverse` for an alternative icon color.
