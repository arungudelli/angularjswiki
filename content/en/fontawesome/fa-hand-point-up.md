## `Hand-Point-Up` font awesome icon CSS class name

To display `Hand-Point-Up` font awesome icon, add predefined class name i.e.,`fa-hand-point-up` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Hand-Point-Up` icon.


 `Hand-Point-Up` font awesome icon has 2 icon styles.
 To Display regular icon use `far` class. 
 To Display solid icon use `fas` class. 

```

<i class='fas fa-hand-point-up'></i>
```
