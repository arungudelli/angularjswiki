## `Cpanel` font awesome icon CSS class name

To display `Cpanel` font awesome icon, add predefined class name i.e.,`fa-cpanel` (with prefix `fa-`) to the `i` tag.

 We need to append icon style class `fab`.

## `Cpanel` font awesome icon with Fixed Width

And use class `fa-inverse` for an alternative icon color.
