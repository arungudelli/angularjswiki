## `Toilet-Paper-Slash` font awesome icon CSS class name

To display `Toilet-Paper-Slash` font awesome icon, add predefined class name i.e.,`fa-toilet-paper-slash` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Toilet-Paper-Slash` icon.


`Toilet-Paper-Slash` icon has 1 icon style i.e.,solid. 
 We need to append icon style class `fas`.

And use class `fa-inverse` for an alternative icon color.
