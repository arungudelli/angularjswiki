## `Zhihu` font awesome icon CSS class name

To display `Zhihu` font awesome icon, add predefined class name i.e.,`fa-zhihu` (with prefix `fa-`) to the `i` tag.

`Zhihu` icon has 1 icon style i.e.,brands. 
 We need to append icon style class `fab`.

## `Zhihu` font awesome icon with Fixed Width
