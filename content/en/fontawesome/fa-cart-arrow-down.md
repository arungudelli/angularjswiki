## `Cart-Arrow-Down` font awesome icon CSS class name

To display `Cart-Arrow-Down` font awesome icon, add predefined class name i.e.,`fa-cart-arrow-down` (with prefix `fa-`) to the `i` tag.

`Cart-Arrow-Down` icon has 1 icon style i.e.,solid. 
 We need to append icon style class `fas`.

And use class `fa-inverse` for an alternative icon color. 

In the below example, I am displaying `Cart-Arrow-Down` font awesome icon on top of icons like `fa-square`,`fa-circle` and `fa-ban`.
