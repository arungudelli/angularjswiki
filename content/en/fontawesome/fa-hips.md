## `Hips` font awesome icon CSS class name

To display `Hips` font awesome icon, add predefined class name i.e.,`fa-hips` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Hips` icon.


`Hips` icon has 1 icon style i.e.,brands. 
 We need to append icon style class `fab`.

## `Hips` font awesome icon with Fixed Width


And use class `fa-inverse` for an alternative icon color.
