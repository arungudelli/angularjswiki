**`Keyboard` font awesome icon css class name is `fa-keyboard` and CSS Content Code is ` \f11c`**.

`Keyboard` font awesome icon has 2 style types i.e.,  regular(far), solid(fas) 


{{%toc%}}
## `Keyboard` font awesome icon usage
We can display `Keyboard` font awesome icon using 3 different ways depending upon our requirement.

1. Using CSS Class Name 

2. Using CSS Content Code 

3. Using SVG 



## `Keyboard` font awesome icon CSS class name

To display `Keyboard` font awesome icon, add predefined class name i.e.,`fa-keyboard` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Keyboard` icon.


 `Keyboard` font awesome icon has 2 icon styles.
 To Display regular icon use `far` class. 
 To Display solid icon use `fas` class. 

```
<i class='far fa-keyboard'></i>
<i class='fas fa-keyboard'></i>

```

## `Keyboard` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Keyboard` icon and `home` icon may not have same height and width.

So to display two icons with fixed width and height we can use `fa-fw` class.
```
<i style='border:1px solid;' class='fas fa-keyboard fa-fw fa-3x'></i>Fixed Width<i style='border:1px solid;' class='fas fa-keyboard fa-3x'></i>Normal<br/>
```

To do that use the `fa-stack` class on the parent, the `fa-stack-1x` for the regularly sized icon, and `fa-stack-2x` for the larger icon.

And use class `fa-inverse` for an alternative icon color.
