## `Mendeley` font awesome icon usage
We can display `Mendeley` font awesome icon using 3 different ways depending upon our requirement.

1. Using CSS Class Name 

2. Using CSS Content Code 

3. Using SVG 



## `Mendeley` font awesome icon CSS class name

To display `Mendeley` font awesome icon, add predefined class name i.e.,`fa-mendeley` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Mendeley` icon.


`Mendeley` icon has 1 icon style i.e.,brands. 
 We need to append icon style class `fab`.
```
<i class='fab fa-mendeley'></i>

```

<div style='border:1px solid rgba(0,0,0,.1);margin-bottom:10px;padding:5px;'>
<p>Output:</p>


<i class='fab fa-mendeley'></i>

</div>

## Change `Mendeley` font awesome icon size
To increase `Mendeley` font awesome icon size, use the `fa-lg(33 % increase)`, `fa-2x`, `fa-3x`, `fa-4x`, or `fa-5x` classes along with icon class  \f7b3.
Increase in icon size will be relative to their parent container.

```
<i class='fab fa-mendeley fa-3x'>fa-3x</i><br/>
```

## `Mendeley` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Mendeley` icon and `home` icon may not have same height and width.

So to display two icons with fixed width and height we can use `fa-fw` class.
```
<i style='border:1px solid;' class='fab fa-mendeley fa-fw fa-3x'></i>Fixed Width<i style='border:1px solid;' class='fab fa-mendeley fa-3x'></i>Normal<br/>
```

## Combine `Mendeley` font awesome icon with other font awesome icons

In font awesome icons,using stacking we can combine multiple icons and display it as one icon.

To do that use the `fa-stack` class on the parent, the `fa-stack-1x` for the regularly sized icon, and `fa-stack-2x` for the larger icon.

And use class `fa-inverse` for an alternative icon color. 

In the below example, I am displaying `Mendeley` font awesome icon on top of icons like `fa-square`,`fa-circle` and `fa-ban`.

And used `fa-inverse` in the second example to invert the color of the icon.
```
<div>
    <span class='fa-stack fa-lg'>
        <i class='far fa-square fa-stack-2x'></i>
        <i class='fab fa-mendeley fa-stack-1x'></i>
    </span>
      Mendeley on fa-square<br>
    <span class='fa-stack fa-lg'>
        <i class='fas fa-circle fa-stack-2x'></i>
        <i class='fab fa-mendeley fa-stack-1x fa-inverse'></i>
    </span>
      Mendeley on fa-circle<br>

    <span class='fa-stack fa-lg'>
        <i class='fab fa-mendeley fa-stack-1x'></i>
        <i class='fas fa-ban fa-stack-2x'></i>
    </span>
      Mendeley on fa-ban
</div>
```
