**`Indent` font awesome icon css class name is `fa-indent` and CSS Content Code is ` \f03c`**.
 

`Indent` font awesome icon is part of solid icons. 



{{%toc%}}
## `Indent` font awesome icon usage
We can display `Indent` font awesome icon using 3 different ways depending upon our requirement.

1. Using CSS Class Name 

2. Using CSS Content Code 

3. Using SVG 



## `Indent` font awesome icon CSS class name

To display `Indent` font awesome icon, add predefined class name i.e.,`fa-indent` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Indent` icon.


`Indent` icon has 1 icon style i.e.,solid. 
 We need to append icon style class `fas`.
```
<i class='fas fa-indent'></i>

```

<p>Output:</p>


<i class='fas fa-indent'></i>

</div>


## `Indent` font awesome icon CSS Content Code 

We can display `Indent` font awesome icon using it's CSS Content Code ` \f03c` 

Use the following HTML code 

```
<li><span class='Indent fontawesomeicon'></span>Indent</li>
```

Apply CSS code 

```
<style> 
.fontawesomeicon::before {
    display: inline-block;
    text-rendering: auto;
    -webkit-font-smoothing: antialiased;
 }

   font: var(--fa-font-solid);

 }</style>
```

## Change `Indent` font awesome icon size
To increase `Indent` font awesome icon size, use the `fa-lg(33 % increase)`, `fa-2x`, `fa-3x`, `fa-4x`, or `fa-5x` classes along with icon class  \f03c.
Increase in icon size will be relative to their parent container.
```
<i class='fas fa-indent fa-lg'>fa-lg</i><br/>
<i class='fas fa-indent fa-2x'>fa-2x</i><br/>
<i class='fas fa-indent fa-3x'>fa-3x</i><br/>
<i class='fas fa-indent fa-4x'>fa-4x</i><br/>
<i class='fas fa-indent fa-5x'>fa-5x</i><br/>
```

## `Indent` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Indent` icon and `home` icon may not have same height and width.

So to display two icons with fixed width and height we can use `fa-fw` class.
```
<i style='border:1px solid;' class='fas fa-indent fa-fw fa-3x'></i>Fixed Width<i style='border:1px solid;' class='fas fa-indent fa-3x'></i>Normal<br/>
<i style='border:1px solid;' class='fas fa-home fa-fw fa-3x'></i>Fixed Width<i style='border:1px solid;' class='fas fa-home fa-3x'></i>Normal<br/>
```

## Combine `Indent` font awesome icon with other font awesome icons

In font awesome icons,using stacking we can combine multiple icons and display it as one icon.

To do that use the `fa-stack` class on the parent, the `fa-stack-1x` for the regularly sized icon, and `fa-stack-2x` for the larger icon.

And use class `fa-inverse` for an alternative icon color. 

In the below example, I am displaying `Indent` font awesome icon on top of icons like `fa-square`,`fa-circle` and `fa-ban`.

And used `fa-inverse` in the second example to invert the color of the icon.
```
<div>
    <span class='fa-stack fa-lg'>
        <i class='far fa-square fa-stack-2x'></i>
        <i class='fas fa-indent fa-stack-1x'></i>
    </span>
      Indent on fa-square<br>
    <span class='fa-stack fa-lg'>
        <i class='fas fa-circle fa-stack-2x'></i>
        <i class='fas fa-indent fa-stack-1x fa-inverse'></i>
    </span>
      Indent on fa-circle<br>

    <span class='fa-stack fa-lg'>
        <i class='fas fa-indent fa-stack-1x'></i>
        <i class='fas fa-ban fa-stack-2x'></i>
    </span>
      Indent on fa-ban
</div>
```
