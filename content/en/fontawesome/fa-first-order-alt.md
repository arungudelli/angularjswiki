## `First-Order-Alt` font awesome icon CSS class name

To display `First-Order-Alt` font awesome icon, add predefined class name i.e.,`fa-first-order-alt` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `First-Order-Alt` icon.


`First-Order-Alt` icon has 1 icon style i.e.,brands. 
 We need to append icon style class `fab`.

And use class `fa-inverse` for an alternative icon color.
