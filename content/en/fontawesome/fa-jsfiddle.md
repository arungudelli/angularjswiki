## `Jsfiddle` font awesome icon usage
We can display `Jsfiddle` font awesome icon using 3 different ways depending upon our requirement.

1. Using CSS Class Name 

2. Using CSS Content Code 

3. Using SVG 



## `Jsfiddle` font awesome icon CSS class name

To display `Jsfiddle` font awesome icon, add predefined class name i.e.,`fa-jsfiddle` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Jsfiddle` icon.


`Jsfiddle` icon has 1 icon style i.e.,brands. 
 We need to append icon style class `fab`.
```
<i class='fab fa-jsfiddle'></i>

```

## `Jsfiddle` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Jsfiddle` icon and `home` icon may not have same height and width.


And use class `fa-inverse` for an alternative icon color.
