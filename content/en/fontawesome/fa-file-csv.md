To display `File-Csv` font awesome icon, add predefined class name i.e.,`fa-file-csv` (with prefix `fa-`) to the `i` tag.

`File-Csv` icon has 1 icon style i.e.,solid. 
 We need to append icon style class `fas`.

And use class `fa-inverse` for an alternative icon color.
