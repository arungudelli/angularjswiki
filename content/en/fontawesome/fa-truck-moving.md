To display `Truck-Moving` font awesome icon, add predefined class name i.e.,`fa-truck-moving` (with prefix `fa-`) to the `i` tag.

## Combine `Truck-Moving` font awesome icon with other font awesome icons

To do that use the `fa-stack` class on the parent, the `fa-stack-1x` for the regularly sized icon, and `fa-stack-2x` for the larger icon.

And use class `fa-inverse` for an alternative icon color. 

In the below example, I am displaying `Truck-Moving` font awesome icon on top of icons like `fa-square`,`fa-circle` and `fa-ban`.

And used `fa-inverse` in the second example to invert the color of the icon.
