## `Hooli` font awesome icon CSS class name

To display `Hooli` font awesome icon, add predefined class name i.e.,`fa-hooli` (with prefix `fa-`) to the `i` tag.

 We need to append icon style class `fab`.

And use class `fa-inverse` for an alternative icon color.
