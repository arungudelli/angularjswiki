## `Award` font awesome icon CSS class name

To display `Award` font awesome icon, add predefined class name i.e.,`fa-award` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Award` icon.


`Award` icon has 1 icon style i.e.,solid. 
 We need to append icon style class `fas`.
```

## `Award` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Award` icon and `home` icon may not have same height and width.

```

And use class `fa-inverse` for an alternative icon color.
