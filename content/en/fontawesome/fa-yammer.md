## `Yammer` font awesome icon CSS class name

To display `Yammer` font awesome icon, add predefined class name i.e.,`fa-yammer` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Yammer` icon.


`Yammer` icon has 1 icon style i.e.,brands. 
 We need to append icon style class `fab`.
```
<i class='fab fa-yammer'></i>

```

## `Yammer` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Yammer` icon and `home` icon may not have same height and width.

So to display two icons with fixed width and height we can use `fa-fw` class.
```
<i style='border:1px solid;' class='fab fa-yammer fa-fw fa-3x'></i>Fixed Width<i style='border:1px solid;' class='fab fa-yammer fa-3x'></i>Normal<br/>
```

And use class `fa-inverse` for an alternative icon color.
