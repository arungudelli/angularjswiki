## `Chess-Rook` font awesome icon CSS class name

To display `Chess-Rook` font awesome icon, add predefined class name i.e.,`fa-chess-rook` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Chess-Rook` icon.


 `Chess-Rook` font awesome icon has 2 icon styles.
 To Display regular icon use `far` class. 
 To Display solid icon use `fas` class.
