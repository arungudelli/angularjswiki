**`Facebook-F` font awesome icon css class name is `fa-facebook-f` and CSS Content Code is ` \f39e`**.
 

`Facebook-F` font awesome icon is part of brands icons.

## `Facebook-F` font awesome icon usage
We can display `Facebook-F` font awesome icon using 3 different ways depending upon our requirement.

1. Using CSS Class Name 

2. Using CSS Content Code 

3. Using SVG 



## `Facebook-F` font awesome icon CSS class name

To display `Facebook-F` font awesome icon, add predefined class name i.e.,`fa-facebook-f` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Facebook-F` icon.


`Facebook-F` icon has 1 icon style i.e.,brands. 
 We need to append icon style class `fab`.
```
<i class='fab fa-facebook-f'></i>

```

To increase `Facebook-F` font awesome icon size, use the `fa-lg(33 % increase)`, `fa-2x`, `fa-3x`, `fa-4x`, or `fa-5x` classes along with icon class  \f39e.

## `Facebook-F` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Facebook-F` icon and `home` icon may not have same height and width.

So to display two icons with fixed width and height we can use `fa-fw` class.
```
<i style='border:1px solid;' class='fab fa-facebook-f fa-fw fa-3x'></i>Fixed Width<i style='border:1px solid;' class='fab fa-facebook-f fa-3x'></i>Normal<br/>
```

## Combine `Facebook-F` font awesome icon with other font awesome icons

In font awesome icons,using stacking we can combine multiple icons and display it as one icon.

To do that use the `fa-stack` class on the parent, the `fa-stack-1x` for the regularly sized icon, and `fa-stack-2x` for the larger icon.

And use class `fa-inverse` for an alternative icon color. 

In the below example, I am displaying `Facebook-F` font awesome icon on top of icons like `fa-square`,`fa-circle` and `fa-ban`.

And used `fa-inverse` in the second example to invert the color of the icon.
```
<div>
    <span class='fa-stack fa-lg'>
        <i class='far fa-square fa-stack-2x'></i>
        <i class='fab fa-facebook-f fa-stack-1x'></i>
    </span>
      Facebook-F on fa-square<br>
    <span class='fa-stack fa-lg'>
        <i class='fas fa-circle fa-stack-2x'></i>
        <i class='fab fa-facebook-f fa-stack-1x fa-inverse'></i>
    </span>
      Facebook-F on fa-circle<br>

    <span class='fa-stack fa-lg'>
        <i class='fab fa-facebook-f fa-stack-1x'></i>
        <i class='fas fa-ban fa-stack-2x'></i>
    </span>
      Facebook-F on fa-ban
</div>
```
