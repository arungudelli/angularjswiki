## `Square-Github` font awesome icon CSS class name

To display `Square-Github` font awesome icon, add predefined class name i.e.,`fa-square-github` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Square-Github` icon.


`Square-Github` icon has 1 icon style i.e.,brands. 
 We need to append icon style class `fab`.
```
<i class='fab fa-square-github'></i>

```
