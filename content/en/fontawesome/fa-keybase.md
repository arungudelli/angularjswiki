## `Keybase` font awesome icon CSS class name

To display `Keybase` font awesome icon, add predefined class name i.e.,`fa-keybase` (with prefix `fa-`) to the `i` tag.

## `Keybase` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Keybase` icon and `home` icon may not have same height and width.

So to display two icons with fixed width and height we can use `fa-fw` class.

And use class `fa-inverse` for an alternative icon color.
