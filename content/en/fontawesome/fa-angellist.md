## `Angellist` font awesome icon CSS class name

To display `Angellist` font awesome icon, add predefined class name i.e.,`fa-angellist` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Angellist` icon.


`Angellist` icon has 1 icon style i.e.,brands. 
 We need to append icon style class `fab`.
```
<i class='fab fa-angellist'></i>

```

## `Angellist` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Angellist` icon and `home` icon may not have same height and width.

So to display two icons with fixed width and height we can use `fa-fw` class.

And use class `fa-inverse` for an alternative icon color.
