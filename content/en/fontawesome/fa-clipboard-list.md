**`Clipboard-List` font awesome icon css class name is `fa-clipboard-list` and CSS Content Code is ` \f46d`**.
 

`Clipboard-List` font awesome icon is part of solid icons.

## `Clipboard-List` font awesome icon usage
We can display `Clipboard-List` font awesome icon using 3 different ways depending upon our requirement.

1. Using CSS Class Name 

2. Using CSS Content Code 

3. Using SVG 



## `Clipboard-List` font awesome icon CSS class name

To display `Clipboard-List` font awesome icon, add predefined class name i.e.,`fa-clipboard-list` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Clipboard-List` icon.


`Clipboard-List` icon has 1 icon style i.e.,solid. 
 We need to append icon style class `fas`.
```
<i class='fas fa-clipboard-list'></i>

```

## Change `Clipboard-List` font awesome icon size
To increase `Clipboard-List` font awesome icon size, use the `fa-lg(33 % increase)`, `fa-2x`, `fa-3x`, `fa-4x`, or `fa-5x` classes along with icon class  \f46d.
Increase in icon size will be relative to their parent container.

```
<i class='fas fa-clipboard-list fa-3x'>fa-3x</i><br/>
```

## `Clipboard-List` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Clipboard-List` icon and `home` icon may not have same height and width.

## Combine `Clipboard-List` font awesome icon with other font awesome icons

In font awesome icons,using stacking we can combine multiple icons and display it as one icon.

To do that use the `fa-stack` class on the parent, the `fa-stack-1x` for the regularly sized icon, and `fa-stack-2x` for the larger icon.

And use class `fa-inverse` for an alternative icon color. 

In the below example, I am displaying `Clipboard-List` font awesome icon on top of icons like `fa-square`,`fa-circle` and `fa-ban`.

And used `fa-inverse` in the second example to invert the color of the icon.
```
<div>
    <span class='fa-stack fa-lg'>
        <i class='far fa-square fa-stack-2x'></i>
        <i class='fas fa-clipboard-list fa-stack-1x'></i>
    </span>
      Clipboard-List on fa-square<br>
    <span class='fa-stack fa-lg'>
        <i class='fas fa-circle fa-stack-2x'></i>

      Clipboard-List on fa-ban
</div>
```
