To display `Arrow-Up-A-Z` font awesome icon, add predefined class name i.e.,`fa-arrow-up-a-z` (with prefix `fa-`) to the `i` tag.

`Arrow-Up-A-Z` icon has 1 icon style i.e.,solid.
