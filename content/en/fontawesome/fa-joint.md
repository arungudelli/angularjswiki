2. Using CSS Content Code

## `Joint` font awesome icon CSS class name

To display `Joint` font awesome icon, add predefined class name i.e.,`fa-joint` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Joint` icon.


`Joint` icon has 1 icon style i.e.,solid. 
 We need to append icon style class `fas`.

## `Joint` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Joint` icon and `home` icon may not have same height and width.

So to display two icons with fixed width and height we can use `fa-fw` class.
```
<i style='border:1px solid;' class='fas fa-joint fa-fw fa-3x'></i>Fixed Width<i style='border:1px solid;' class='fas fa-joint fa-3x'></i>Normal<br/>
<i style='border:1px solid;' class='fas fa-home fa-fw fa-3x'></i>Fixed Width<i style='border:1px solid;' class='fas fa-home fa-3x'></i>Normal<br/>
```

## Combine `Joint` font awesome icon with other font awesome icons

In font awesome icons,using stacking we can combine multiple icons and display it as one icon.

To do that use the `fa-stack` class on the parent, the `fa-stack-1x` for the regularly sized icon, and `fa-stack-2x` for the larger icon.

And use class `fa-inverse` for an alternative icon color. 

In the below example, I am displaying `Joint` font awesome icon on top of icons like `fa-square`,`fa-circle` and `fa-ban`.

And used `fa-inverse` in the second example to invert the color of the icon.
