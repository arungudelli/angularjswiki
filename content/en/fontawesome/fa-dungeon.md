## `Dungeon` font awesome icon CSS class name

To display `Dungeon` font awesome icon, add predefined class name i.e.,`fa-dungeon` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Dungeon` icon.


`Dungeon` icon has 1 icon style i.e.,solid. 
 We need to append icon style class `fas`.
```
<i class='fas fa-dungeon'></i>

```

And use class `fa-inverse` for an alternative icon color.
