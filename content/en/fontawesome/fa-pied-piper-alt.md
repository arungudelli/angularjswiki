2. Using CSS Content Code 

3. Using SVG 



## `Pied-Piper-Alt` font awesome icon CSS class name

To display `Pied-Piper-Alt` font awesome icon, add predefined class name i.e.,`fa-pied-piper-alt` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Pied-Piper-Alt` icon.


`Pied-Piper-Alt` icon has 1 icon style i.e.,brands. 
 We need to append icon style class `fab`.
```
<i class='fab fa-pied-piper-alt'></i>

```
