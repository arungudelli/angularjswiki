## `500Px` font awesome icon CSS class name

To display `500Px` font awesome icon, add predefined class name i.e.,`fa-500px` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `500Px` icon.


`500Px` icon has 1 icon style i.e.,brands. 
 We need to append icon style class `fab`.
```
<i class='fab fa-500px'></i>

```

## `500Px` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `500Px` icon and `home` icon may not have same height and width.

## Combine `500Px` font awesome icon with other font awesome icons

In font awesome icons,using stacking we can combine multiple icons and display it as one icon.

To do that use the `fa-stack` class on the parent, the `fa-stack-1x` for the regularly sized icon, and `fa-stack-2x` for the larger icon.

And use class `fa-inverse` for an alternative icon color. 

In the below example, I am displaying `500Px` font awesome icon on top of icons like `fa-square`,`fa-circle` and `fa-ban`.

And used `fa-inverse` in the second example to invert the color of the icon.
```

    </span>
      500Px on fa-square<br>
```
