## `Yarn` font awesome icon usage
We can display `Yarn` font awesome icon using 3 different ways depending upon our requirement.

1. Using CSS Class Name 

2. Using CSS Content Code 

3. Using SVG 



## `Yarn` font awesome icon CSS class name

To display `Yarn` font awesome icon, add predefined class name i.e.,`fa-yarn` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Yarn` icon.


`Yarn` icon has 1 icon style i.e.,brands. 
 We need to append icon style class `fab`.
```
<i class='fab fa-yarn'></i>

```

## `Yarn` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Yarn` icon and `home` icon may not have same height and width.

So to display two icons with fixed width and height we can use `fa-fw` class.
```
<i style='border:1px solid;' class='fab fa-yarn fa-fw fa-3x'></i>Fixed Width<i style='border:1px solid;' class='fab fa-yarn fa-3x'></i>Normal<br/>
<i style='border:1px solid;' class='fas fa-home fa-fw fa-3x'></i>Fixed Width<i style='border:1px solid;' class='fas fa-home fa-3x'></i>Normal<br/>

```

## Combine `Yarn` font awesome icon with other font awesome icons

In font awesome icons,using stacking we can combine multiple icons and display it as one icon.

To do that use the `fa-stack` class on the parent, the `fa-stack-1x` for the regularly sized icon, and `fa-stack-2x` for the larger icon.

And use class `fa-inverse` for an alternative icon color. 

In the below example, I am displaying `Yarn` font awesome icon on top of icons like `fa-square`,`fa-circle` and `fa-ban`.

And used `fa-inverse` in the second example to invert the color of the icon.
