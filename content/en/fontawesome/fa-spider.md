## `Spider` font awesome icon CSS class name

To display `Spider` font awesome icon, add predefined class name i.e.,`fa-spider` (with prefix `fa-`) to the `i` tag.

`Spider` icon has 1 icon style i.e.,solid. 
 We need to append icon style class `fas`.

## `Spider` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Spider` icon and `home` icon may not have same height and width.

So to display two icons with fixed width and height we can use `fa-fw` class.
