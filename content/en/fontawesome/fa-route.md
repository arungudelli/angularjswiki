## `Route` font awesome icon CSS class name

To display `Route` font awesome icon, add predefined class name i.e.,`fa-route` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Route` icon.


`Route` icon has 1 icon style i.e.,solid. 
 We need to append icon style class `fas`.
```
<i class='fas fa-route'></i>

```

## `Route` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Route` icon and `home` icon may not have same height and width.

So to display two icons with fixed width and height we can use `fa-fw` class.
```
<i style='border:1px solid;' class='fas fa-route fa-fw fa-3x'></i>Fixed Width<i style='border:1px solid;' class='fas fa-route fa-3x'></i>Normal<br/>
```

## Combine `Route` font awesome icon with other font awesome icons

In font awesome icons,using stacking we can combine multiple icons and display it as one icon.

To do that use the `fa-stack` class on the parent, the `fa-stack-1x` for the regularly sized icon, and `fa-stack-2x` for the larger icon.

And use class `fa-inverse` for an alternative icon color. 

In the below example, I am displaying `Route` font awesome icon on top of icons like `fa-square`,`fa-circle` and `fa-ban`.

And used `fa-inverse` in the second example to invert the color of the icon.
```
<div>
    <span class='fa-stack fa-lg'>
        <i class='far fa-square fa-stack-2x'></i>
        <i class='fas fa-route fa-stack-1x'></i>
    </span>
      Route on fa-square<br>
    <span class='fa-stack fa-lg'>
        <i class='fas fa-circle fa-stack-2x'></i>
        <i class='fas fa-route fa-stack-1x fa-inverse'></i>
    </span>
      Route on fa-circle<br>

        <i class='fas fa-route fa-stack-1x'></i>
        <i class='fas fa-ban fa-stack-2x'></i>

</div>
```
