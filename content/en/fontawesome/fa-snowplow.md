## `Snowplow` font awesome icon CSS class name

To display `Snowplow` font awesome icon, add predefined class name i.e.,`fa-snowplow` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Snowplow` icon.


`Snowplow` icon has 1 icon style i.e.,solid. 
 We need to append icon style class `fas`.
```
<i class='fas fa-snowplow'></i>

```

## `Snowplow` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Snowplow` icon and `home` icon may not have same height and width.

So to display two icons with fixed width and height we can use `fa-fw` class.
```

## Combine `Snowplow` font awesome icon with other font awesome icons

In font awesome icons,using stacking we can combine multiple icons and display it as one icon.

To do that use the `fa-stack` class on the parent, the `fa-stack-1x` for the regularly sized icon, and `fa-stack-2x` for the larger icon.

And use class `fa-inverse` for an alternative icon color. 

In the below example, I am displaying `Snowplow` font awesome icon on top of icons like `fa-square`,`fa-circle` and `fa-ban`.

And used `fa-inverse` in the second example to invert the color of the icon.
```
<div>
    <span class='fa-stack fa-lg'>
        <i class='far fa-square fa-stack-2x'></i>
        <i class='fas fa-snowplow fa-stack-1x'></i>
    </span>
      Snowplow on fa-square<br>
