We can display `Gitkraken` font awesome icon using 3 different ways depending upon our requirement.

2. Using CSS Content Code 

3. Using SVG 



## `Gitkraken` font awesome icon CSS class name

To display `Gitkraken` font awesome icon, add predefined class name i.e.,`fa-gitkraken` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Gitkraken` icon.


`Gitkraken` icon has 1 icon style i.e.,brands. 
 We need to append icon style class `fab`.
```
<i class='fab fa-gitkraken'></i>

```

## `Gitkraken` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

And use class `fa-inverse` for an alternative icon color.
