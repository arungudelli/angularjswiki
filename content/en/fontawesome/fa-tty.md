## `Tty` font awesome icon CSS class name

To display `Tty` font awesome icon, add predefined class name i.e.,`fa-tty` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Tty` icon.


`Tty` icon has 1 icon style i.e.,solid. 
 We need to append icon style class `fas`.
```
<i class='fas fa-tty'></i>

```

## `Tty` font awesome icon with Fixed Width
All the font awesome icons does not have same width.

For example `Tty` icon and `home` icon may not have same height and width.

So to display two icons with fixed width and height we can use `fa-fw` class.

And use class `fa-inverse` for an alternative icon color.
