## `Hand-Point-Down` font awesome icon CSS class name

To display `Hand-Point-Down` font awesome icon, add predefined class name i.e.,`fa-hand-point-down` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Hand-Point-Down` icon.


 `Hand-Point-Down` font awesome icon has 2 icon styles.
 To Display regular icon use `far` class. 
 To Display solid icon use `fas` class. 

```
<i class='far fa-hand-point-down'></i>
<i class='fas fa-hand-point-down'></i>

```

And use class `fa-inverse` for an alternative icon color.
