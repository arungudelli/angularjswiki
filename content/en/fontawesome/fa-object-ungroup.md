## `Object-Ungroup` font awesome icon CSS class name

To display `Object-Ungroup` font awesome icon, add predefined class name i.e.,`fa-object-ungroup` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Object-Ungroup` icon.


 `Object-Ungroup` font awesome icon has 2 icon styles.
 To Display regular icon use `far` class. 
 To Display solid icon use `fas` class.

And use class `fa-inverse` for an alternative icon color.
