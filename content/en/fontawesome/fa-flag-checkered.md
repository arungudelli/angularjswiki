## `Flag-Checkered` font awesome icon CSS class name

To display `Flag-Checkered` font awesome icon, add predefined class name i.e.,`fa-flag-checkered` (with prefix `fa-`) to the `i` tag. 

And we need to add corresponding font awesome icon style for the `Flag-Checkered` icon.


`Flag-Checkered` icon has 1 icon style i.e.,solid. 
 We need to append icon style class `fas`.
```
<i class='fas fa-flag-checkered'></i>

```
